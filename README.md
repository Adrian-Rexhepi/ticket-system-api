Ticketsystem REST API

Dieses Projekt ist eine REST API für ein Ticketsystem, entwickelt in Java. Es ermöglicht die Verwaltung von Tickets, z.B. für Support-Anfragen, Bug-Tracking oder allgemeine Aufgabenverwaltung. Die API unterstützt grundlegende CRUD-Operationen (Create, Read, Update, Delete) für Tickets und bietet eine einfache Schnittstelle, um Tickets programmgesteuert zu verwalten.

Funktionen:

Ticket erstellen: Neue Tickets anlegen mit Titel, Beschreibung, Priorität, Status und weiteren Details.
Tickets abrufen: Alle Tickets oder einzelne Tickets nach ID abrufen.
Tickets aktualisieren: Bestehende Tickets ändern (z.B. Status, Priorität).
Tickets löschen: Nicht mehr benötigte Tickets entfernen.
Filter und Suche: Tickets nach Status, Priorität oder anderen Kriterien filtern.

Technologien:

Java 21
Spring Boot
REST-Architektur
JSON
MySQL
